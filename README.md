## Como colaborar 🔐
1. Clonando:
-  *git clone https://github.com/petersilvahs/BonsDrinks.git*
2. No terminal do computador ou no VS Code esteja instalado: 
- *npm ou yarn install*
3. Para rodar, utilize:
- *npm ou yark install*
4. Acesse o site:
- *O site abrirá na sua aba padrão na porta localhost: 3000*
